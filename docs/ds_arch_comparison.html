Direction-Speed Architecture

Post-hoc 是外部坐标转换，Bijector 是 actor 内部分布变换

两者都想把位移拆成方向和速度，但发生的位置完全不同：post-hoc 在数据到模型之间做 decompose，在模型到环境之间做 compose；bijector 只在 actor 概率分布内部把 5D latent action 双向变换为 5D env action。

Post-hoc: 模型看到 6D dataset/env 是 5D，但 actor 和 critic 学习的是 decomposed 6D 动作。
Bijector: 外部始终 5D dataset、critic、actor output、env action 都是 5D；只有分布内部有 latent → output。
